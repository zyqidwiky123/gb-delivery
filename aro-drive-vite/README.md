# GB Delivery PWA (Customer App)

A modern, fast, and responsive Progressice Web App (PWA) for delivery services in Blitar. Built with React, Vite, and Firebase.

## 🚀 Fitur Utama
- **Real-time Order Tracking**: Lacak pesanan Anda secara langsung.
- **Loyalty Points System**: Kumpulkan poin dari setiap pesanan dan tukarkan dengan voucher.
- **PWA Ready**: Dapat diinstal di HP (Android/iOS) dan berfungsi seperti aplikasi native.
- **Role-based Access**: Terintegrasi dengan Admin Dashboard dan Driver App.

## 🛠️ Stack Teknologi
- **Frontend**: React.js 19
- **Build Tool**: Vite 6
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Backend & Auth**: Firebase (Firestore, Authentication)
- **Maps**: Leaflet / React-Leaflet
- **PWA Plugin**: Vite-plugin-pwa

## 📦 Instalasi & Penggunaan Lokal

1. **Clone Repository**:
   ```bash
   git clone <url-repository-anda>
   cd gb-delivery-vite
   ```

2. **Install Dependensi**:
   ```bash
   npm install
   ```

3. **Jalankan Aplikasi Mode Dev**:
   ```bash
   npm run dev
   ```

4. **Build untuk Produksi**:
   ```bash
   npm run build
   ```

## 📄 Struktur Folder
- `src/`: Source code utama (components, pages, hooks, firebase config).
- `public/`: Aset statis dan konfigurasi PWA.
- `dist/`: Hasil build produksi (otomatis di-generate).

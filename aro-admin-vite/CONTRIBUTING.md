# Kontribusi ke gb-admin

Terima kasih telah tertarik untuk berkontribusi pada proyek **gb-admin**! Proyek ini adalah dashboard administrasi untuk sistem GB Delivery Blitar.

## Cara Berkontribusi

1.  **Fork** repository ini.
2.  Buat **Branch** baru untuk fitur atau perbaikan bug Anda (`git checkout -b feature/nama-fitur`).
3.  Lakukan **Commit** perubahan Anda (`git commit -m 'Menambahkan fitur XYZ'`).
4.  **Push** ke branch tersebut (`git push origin feature/nama-fitur`).
5.  Buka **Pull Request**.

## Persyaratan Kontribusi

*   Pastikan kode mengikuti standar React 19 dan Tailwind CSS.
*   Selalu jalankan `npm run build` untuk memastikan tidak ada error sebelum kontribusi.
*   Jangan membagikan kredensial Firebase atau file `.env`.

## Melaporkan Masalah

Jika Anda menemukan bug atau memiliki saran fitur, silakan buka **Issue** di halaman GitHub proyek ini.

# GB Delivery - Admin Dashboard
![Build Check](https://github.com/zyqidwiky123/gb-admin/actions/workflows/ci.yml/badge.svg)

Official administration dashboard for GB Delivery Blitar. This app allows administrators to manage orders, drivers, customers, and system settings.

## 🚀 Fitur Utama
- **Order Monitoring**: Pantau status semua pesanan secara real-time.
- **Driver Management**: Kelola data mitra driver dan status aktivasi.
- **System Settings**: Atur parameter sistem, biaya pengiriman, dan poin loyalitas.
- **Analytics**: Lihat ringkasan transaksi dan pendapatan sistem.

## 🛠️ Stack Teknologi
- **Frontend**: React.js 19
- **Build Tool**: Vite 6
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Backend**: Firebase Firestore & Auth

## 📦 Instalasi Lokal

1. **Clone & Masuk ke Folder**:
   ```bash
   git clone https://github.com/zyqidwiky123/gb-admin.git
   cd gb-admin
   ```

2. **Install Dependensi**:
   ```bash
   npm install
   ```

3. **Jalankan Aplikasi**:
   ```bash
   npm run dev
   ```

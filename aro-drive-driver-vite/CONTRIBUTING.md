# Kontribusi ke gb-driver

Terima kasih telah tertarik untuk berkontribusi pada proyek **gb-driver**! Kami menyambut baik bantuan dari siapa saja untuk meningkatkan kualitas aplikasi ini.

## Cara Berkontribusi

1.  **Fork** repository ini.
2.  Buat **Branch** baru untuk fitur atau perbaikan bug Anda (`git checkout -b feature/nama-fitur`).
3.  Lakukan **Commit** perubahan Anda (`git commit -m 'Menambahkan fitur XYZ'`).
4.  **Push** ke branch tersebut (`git push origin feature/nama-fitur`).
5.  Buka **Pull Request**.

## Persyaratan Kontribusi

*   Pastikan kode mengikuti standar yang ada (Tailwind CSS, React Functional Components).
*   Jangan menyertakan file konfigurasi sensitif (seperti `.env`).
*   Pastikan aplikasi dapat dijalankan (`npm run build`) sebelum melakukan PR.

## Melaporkan Masalah

Jika Anda menemukan bug atau memiliki saran fitur, silakan buka **Issue** di halaman GitHub proyek ini.

# GB Delivery - Driver App
![Build Check](https://github.com/zyqidwiky123/gb-driver/actions/workflows/ci.yml/badge.svg)

Official driver application for GB Delivery Blitar. This app allows drivers to receive, manage, and track orders in real-time.

## 🚀 Fitur Utama
- **Order Management**: Terima atau tolak pesanan masuk secara real-time.
- **GPS Navigation**: Navigasi langsung ke lokasi pelanggan menggunakan peta terintegrasi.
- **Earnings Tracker**: Pantau pendapatan harian dan riwayat pesanan.
- **PWA Optimized**: Bekerja maksimal di perangkat Android dan iOS sebagai aplikasi mandiri.

## 🛠️ Stack Teknologi
- **Frontend**: React.js 19
- **Build Tool**: Vite 6
- **Styling**: Tailwind CSS
- **Maps**: Leaflet / React-Leaflet
- **Backend**: Firebase Firestore & Auth
- **PWA**: Vite-plugin-pwa

## 📦 Instalasi Lokal

1. **Clone & Masuk ke Folder**:
   ```bash
   git clone https://github.com/zyqidwiky123/gb-driver.git
   cd gb-driver
   ```

2. **Install Dependensi**:
   ```bash
   npm install
   ```

3. **Jalankan Aplikasi**:
   ```bash
   npm run dev
   ```
